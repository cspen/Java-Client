• Sort Table by Column

    To sort the table by a column, click on the column. The first click sorts ascending, second click sorts descending.

  • Edit a Row

    To edit a row, click the cell you would like to edit. If the cell is editable either enter text, click the checkbox, or select an item from a dropdown list. When entering text, press the enter button or click on another cell to complete the operation.

  • Create a New Row

    To create a new row, click the File menu, then click New. The New Employee dialog box appears. Fill in the form on the box and click OK. The new row will be created on the server and inserted into the top position on the table.

  • Delete a Row

    To delete a row, first click the row you wish to delete. The row will be highlighted blue. Now click the File menu and choose Delete. A dialog box will ask you to confirm before deleting the row from both the server and the table.

  • Refresh Table

    To request the table be refilled with data from the server, click the File menu and choose Refresh.
